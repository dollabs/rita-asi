<h1>What you see</h1>

Genesis analyzes modulation in a story and delivers a modulation report.

<p>The program will read two short article-like stories describing the same conflict from two different perspectives. Based on each author's modulation of the story, such as using passive voice or emphasizing childhood, the program forms an argument about how each author casts blame. 
</p> 
<p>
The program uses a system of experts, each responsible for finding and interpreting a different form of modulation.
</p>
